Hej

Programmering handler om at styre strøm og fortælle computeren hvor den skal sende strømmen hen. Det lyder kompliceret, men det er det ikke. Ikke endnu i hvert fald. Strøm har to muligheder: tænd og sluk, 1 eller 0, sandt eller falsk, ja eller nej. Derfor er meget af logikken sand- eller falsk-logik. Enten er noget sandt eller falsk. Mere om det senere.
